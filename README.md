# smaller

Creates `<skin element>.png` from `<skin element>@2x.png` for every `@2x.png` element in an osu! skin. Supports both `jpg` and `png`, outputs `png`.

## Usage:
```console
$ smaller <path to osu! skin>
```

## Building

POSIX:
```console
$ ./build.sh
```

Windows (requires `MinGW/Clang`)
```console
> ./build.bat
```
